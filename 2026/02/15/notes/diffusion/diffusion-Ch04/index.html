两类扩散模型了解

朗之万动力学去噪分数匹配（SMLD）

我们设定扰动核 \(p_{\sigma}(\tilde{x}|x) := \mathcal{N}(\tilde{x}; x, \sigma^2\mathbf{I})\)，以及扰动（加噪）后的数据边缘概率分布为 \(p_\sigma(\tilde{x}) := \int p_\sigma(\tilde{x}|x) p_\text{data}(x)\text{d}x\)。

考虑一个噪声序列 \(\sigma_\text{min} = \sigma_1 < \sigma_2 < \dots < \sigma_N = \sigma_\text{max}\)，这也就是说每一步的加噪会越来越大。其中，\(\sigma_\text{min}\) 足够小，能够使得 \(p_{\sigma_\text{min}} \approx p_\text{data}(x)\)，\(\sigma_\text{max}\) 足够大，能够使得 \(p_{\sigma_\text{max}} \approx \mathcal{N}(x; 0, \sigma_\text{max}^2\mathbf{I})\)。我们可以训练一个噪声条件分数网络（Noise Conditional Score Network, NCSN），即 \(s_\theta(x, \sigma)\)，使得去噪分数匹配目标 \(\ref{eq:ncsn_objective}\) 最小。 \[ \begin{equation} \theta^* = \arg \min_\theta \sum_{i=1}^N \sigma_i^2 \mathbb{E}_{p_\text{data}(x)} \mathbb{E}_{p_{\sigma_i}(\tilde{x}|x)} [ \| s_\theta(\tilde{x}, \sigma_i) - \textcolor{red}{\nabla_{\tilde{x}} \log p_{\sigma_i}(\tilde{x}|x)} \|_2^2 ] \label{eq:ncsn_objective} \end{equation} \]

只要给定的样本数据足够多，我们就能使用神经网络学习到对于 \(\nabla_{\tilde{x}} \log p_{\sigma_i}(\tilde{x}|x)\) 的较为准确的估计 \(s_\theta(x, \sigma)\)。有了 \(s_\theta(x, \sigma)\)，通过朗之万迭代式 \(\ref{eq:langevin_mcmc}\) 就可以得到前面步的

\[ \begin{equation} \mathbf{x}_i^m = \mathbf{x}_i^{m-1} + \underbrace{\epsilon_i \mathbf{s}_{\theta^*}(\mathbf{x}_i^{m-1}, \sigma_i)}_{\text{1. 爬坡 (Drift)}} + \underbrace{\sqrt{2\epsilon_i} \mathbf{z}_i^m}_{\text{2. 扰动 (Noise)}} \label{eq:langevin_mcmc} \end{equation} \]

这个公式由两股“力量”组成：

  1. 爬坡 (Drift / Gradient Ascent)\(\mathbf{s}_{\theta^*}(\dots)\) 是神经网络预测的分数 (Score)，也就是概率密度的梯度 \(\nabla \log p(x)\)。作用：它告诉数据点：“往这边走！这边的概率更高，更像真实的图片！”\(\epsilon_i\) 是步长 (Step Size)，控制这一步走多远。
  2. 随机扰动 (Noise Injection)\(\mathbf{z}_i^m\) 是从标准正态分布采样出的随机噪声。作用：这是郎之万动力学的精髓。如果只爬坡，容易卡在局部的坑里出不来。加上这个噪声，就像给粒子一点“热运动”，让它有机会跳出局部最优解，去探索更好的区域。系数 \(\sqrt{2\epsilon_i}\) 是物理学推导出来的固定比例，保证粒子最终能稳定在目标分布上。

这是一个双层循环的过程，被称为“退火 (Annealing)”。

  • 外层循环 (\(i = N \to 1\))：噪声等级的接力开始 (\(i=N\))：从巨大的噪声 \(\sigma_{max}\) 开始（此时图片完全是一团糟）。我们先在这个大噪声级别下，跑 \(M\) 步郎之万迭代（内层循环）。在大噪声下，梯度指引的是宏观方向。交接：当第 \(N\) 级跑完 \(M\) 步后，把得到的结果 \(\mathbf{x}_N^M\) 作为下一级 \(N-1\) 的起点 \(\mathbf{x}_{N-1}^0\)。当 \(i=1\) 时，噪声越来越小 (\(\sigma_{min}\))，梯度指引变得越来越精细（开始雕刻头发、纹理），直到最后输出清晰图片。
  • 内层循环 (\(m = 1 \to M\))：在同一级噪声下混合在每一个固定的噪声等级 \(\sigma_i\) 下，我们都要重复执行公式 (2) \(M\) 次。这是为了确保数据点真正“收敛”到了当前噪声等级下的分布 \(p_{\sigma_i}(x)\)。

去噪扩散概率模型（DDPM）

这一部分已经很熟悉了，因此简单讨论一下即可。DDPM 设定了一个离散的马尔可夫链 \({x_0, x_1, \dots, x_N}\)，并且假设了 \(p(x_i | x_{i-1}) = \mathcal{N}(x_i; \sqrt{1 - \beta_i}x_{i-1}, \beta_i\mathbf{I})\)，并且由此可以得到汇总步 \(p_{\alpha_i}(x_i | x_0) = \mathcal{N}(x_i; \underbrace{\sqrt{\alpha_i}x_0}_{\text{均值}}, \underbrace{(1-\alpha_i)I}_{\text{方差}})\)。通过变分下界（ELBO），以分数函数的形式写出来，也就是公式 \(\ref{eq:ddpm_objective}\)。 \[ \begin{equation} \theta^* = \arg \min_\theta \sum_{i=1}^N (1-\alpha_i) \mathbb{E}_{p_{\text{data}}(x)} \mathbb{E}_{p_{\alpha_i}(\tilde{x}|x)} [ \| \mathbf{s}_\theta(\tilde{x}, i) - \underbrace{\nabla_{\tilde{x}} \log p_{\alpha_i}(\tilde{x}|x)}_{\text{真实分数}} \|_2^2 ] \label{eq:ddpm_objective} \end{equation} \]

既然我们训练好了模型 \(s_\theta\)，就可以倒着走，从纯噪声 \(x_N\) 一步步还原回 \(x_0\)。这个过程叫 Ancestral Sampling (祖先采样)，即公式 \(\ref{eq:ancestral_sampling}\) \[ \begin{equation} x_{i-1} = \frac{1}{\sqrt{1-\beta_i}} (x_i + \beta_i \mathbf{s}_{\theta^*}(x_i, i)) + \sqrt{\beta_i} \mathbf{z}_i, \quad i = N, N-1, \dots, 1 \label{eq:ancestral_sampling} \end{equation} \]

  • \(\frac{1}{\sqrt{1-\beta_i}}\) (放大)：因为前向过程把图像缩小了，逆向过程就要把它放大回来。
  • \(\beta_i \mathbf{s}_{\theta^*}\) (去噪/漂移)：利用神经网络预测的 Score（梯度方向），把图片往“清晰”的方向推一把。
  • \(\sqrt{\beta_i} \mathbf{z}_i\) (随机扰动)：加入新的随机噪声 \(\mathbf{z}_i\)。这非常关键！ 如果没有这一项，生成的图片纹理会很假。这和上一张图 SMLD 里的郎之万动力学中的噪声项是一样的作用。

SMLD 是“连续微分方程”的离散化求解；DDPM 是“离散马尔可夫链”的逆向过程。虽然 DDPM 看起来又是马尔可夫链又是变分推断，但把它剥皮拆骨后，它本质上依然是一个加权的去噪分数匹配 (Weighted Denoising Score Matching)。

❓怎么推导的

DDPM 和 SMLD 的真正区别就在于每一级噪声的权重不同。

SMLD (Eq. 1)：权重是 \(\sigma_i^2\)。来源：\(\sigma_i^2 \propto 1 / \mathbb{E}[\|\nabla_{\mathbf{x}} \log p_{\sigma_i}(\tilde{\mathbf{x}} | \mathbf{x})\|_2^2]\)。这意味着 SMLD 的权重是根据 Fisher 信息量来归一化的。

DDPM (Eq. 3)：权重是 \((1-\alpha_i)\)。来源：\((1-\alpha_i) \propto 1 / \mathbb{E}[\|\nabla_{\mathbf{x}} \log p_{\alpha_i}(\tilde{\mathbf{x}} | \mathbf{x})\|_2^2]\)。

具体的表达式不同，但这两个权重在函数形式上是一致的（都反比于 Score 的模长期望）。
