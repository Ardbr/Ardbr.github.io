如何在一个函数中返回多个值

  在 C++ 中，一个函数只能返回一个数据类型，例如返回一个 int 或返回一个 std::string，那如果我们想要返回多个数据，例如一个同时返回一人的名字 std::string 类型，同时 Ta 的年龄 int 类型呢？至少有以下几种方式可以做到在 C++ 中返回多个数据：

  • 将输出作为 ”传出参数“ 以指针或者引用的方式传进这个函数中。这也是许多 C 库函数以及 Unix 系统调用的做法，例如创建一个管道 int pipe(int fd[2])，其函数返回值是 int，-1 表示创建失败，而 0 表示创建成功。而创建成功的话写端就是 fd[1]，读端就是 fd[0]。这两个 fd 就是作为传出参数进入函数中，赋完值之后再在后续进行使用，因此也是函数的返回值的一部分。

函数的参数可以分为 “传入参数” 和 “传出参数”。“传入参数” 仅仅只是为函数正常执行提供必要的数据，通常直接是数据类型，或者为了避免拷贝使用 const & 形式。这种参数仅在内部使用，不会在函数返回的后续继续使用。而 “传出参数” 则是要在函数执行过程中得到所需的值并保存起来，后续要在函数返回后使用这个值，可以说函数的执行大部分目的就是为了得到值并把它赋给对应的传出参数。因此，传出参数多是以指针或者引用的方式传进函数内部。

  • 使用数组或者 vector 或者 array 等，将多个返回值塞进一个容器中整体返回。这种的弊端是所有的返回值的类型必须相同，才能够塞进一个容器中。

  • 使用 pair 或者 tuple。这种比较常见，相比于上面的优点是数据可以是不同的类型了，例如如下的代码：

  但是缺点是访问起来较为麻烦，要使用 std::get<index>(pair | tuple) 进行访问。对于 pair 由于只有两个元素，因此也可以使用 .first 以及 .second 访问。虽然也可以使用 std::tie，但也比较麻烦。但后续的新特性 C++17 中引入了结构化绑定 (structed binding)，这个用起来类似于 Python 的多返回值处理，还是比较方便的。

  • 使用结构体。我们还可以将所有要返回的值构造一个结构体，之后返回的时候只需要用相应的值填好一个结构体，然后返回结构体即可。这种方法的好处是相比于前面的这种靠索引的方法相比更加直观，我可以清晰的知道我在用的是什么，例如 ret.name 说明我正在用 name 的数据。放在前面使用索引，我可能是 std::get<0>(ret)，我必须要记住参数顺序，索引 0 表示的是 name。
