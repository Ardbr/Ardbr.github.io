常用的快捷键

快捷键 功能
Ctrl + p 显示上一条命令，同方向键上
Ctrl + n 显示下一条命令，同方向键下
Ctrl + b 光标向后移动一格，同方向键左
Ctrl + f 光标向前移动一格，同方向键右
Ctrl + a 将光标置于最开始的位置
Ctrl + e 将光标置于末尾的位置
Ctrl + u 清空当前写下的命令
Ctrl + d 删除后面的一个字符，同 delete
